# Remove Duplicates from Sorted Array

## Description

Remove duplicates from sorted array **in place**!

### Example 1:
```
Input: nums = [1,1,2]
Output: 2 = [1,2]
```
### Example 2:
```
Input: nums = [0,0,1,1,1,2,2,3,3,4]
Output: 5 = [0,1,2,3,4]
```
### Constraints:
```
- 1 <= nums.length <= 3 * 104
- -100 <= nums[i] <= 100
- nums is sorted in non-decreasing order.
```
